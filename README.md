## 👋 About Me  
안녕하세요! 저는 최신 공간정보 기술과 **MSA 아키텍처**에 열정을 가진 공간정보 API 서버 개발자입니다.  
**HDMap**과 **SDMap** 같은 최신 포맷을 다루며, **OpenLayers**, **Geotools**, **PostGIS**를 활용한 프로젝트를 성공적으로 이끈 경험이 있습니다.  
대규모 **Map 연산 처리 API 서버 설계**부터 **Map Editor 개발**, **CI/CD 환경 구축**까지 다양한 분야에서 성과를 냈습니다.  
끊임없이 변화하는 기술 트렌드에 발맞추어 성장하며, 조직에 실질적인 가치를 더하는 개발자로 도약 중입니다.
